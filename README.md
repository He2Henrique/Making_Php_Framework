# What is the porpose of this project 
is purely educational to learn how frameworks work and make it as simple as possible.

## How to run this app
- you must have Composer intalled
- use this script:

`` composer install ``
than will the composer json will be initialized the vendor folder.

## autoload system
Inherited classes from the core are preferably loaded outside of the framework, you can se more in composer json to understand what i mean
